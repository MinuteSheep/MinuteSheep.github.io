二、Python数学方法

说起数学方法，肯定不再是简单的加减乘除乘方等基础运算了，数学方法更多是取绝对值、指数运算、对数运算、幂运算，开方运算以及三角函数等，这些方法涉及到的领域也是相当广阔，这部分也是非常重要的，每个人都应该熟练应用，尤其是想要去做数据方面工作的小伙伴。
